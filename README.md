# ToDo App

Einfache Aufgabenverwaltungs-App im Browser entwickelt mit HTML, CSS und JavaScript.

## Funktionen:
- Neue Aufgaben hinzufügen
- Aufgaben als erledigt markieren
- Aufgaben löschen

## Technologien:
- HTML
- CSS
- JavaScript (DOM Manipulation)
